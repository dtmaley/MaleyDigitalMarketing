Powered by the Kentico Headless CMS

This template is built using .NET Angular SPA App template and uses the items below.

  • ASP.NET Core 2.0 and C# for cross-platform server-side code
  • Angular 4 and TypeScript for client-side code
  • Webpack for building and bundling client-side resources
  • Bootstrap 4 (beta) for layout and styling
  • node-SASS for CSS preprocessing
  • angular2-toaster for client side user feedback
  • ng-Bootstrap for Angular Powered Boostrap Components

To fetch our data from Kentico Cloud we are using the Kentico Cloud Boilerplate
